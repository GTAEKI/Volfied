# Volfied

## 👨🏻‍💻 제작 동기
4주간 배운 C# 기본 문법과 게임 제작을 통해 친해지기 위해 제작하였습니다.

## 🖥️ 프로젝트 소개
제작 게임 : Volfied(1989) 모작
게임 소개 : 보스의 반대편 땅을 점령하여 80% 이상 점령시 승리합니다.

## 🕰️ 개발 기간
* 23.06.26(월) - 23.06.30(금)

### 🧑‍🤝‍🧑 맴버구성
 - 배경택
   
### ⚙️ 개발 환경
- Visual Studio for mac
- .Net 7.0
- C#
- Newtonsoft.Json 13.0.3
